# GENZEST AI

### Delhi's AI-Powered Bridal Beauty Booking Platform

Genzest AI is a startup-style bridal beauty marketplace designed to simplify the wedding preparation journey for brides in Delhi. Instead of spending weeks searching for makeup artists, hairstylists, mehendi experts, and beauty professionals across multiple platforms, users can discover verified professionals, generate personalized bridal preparation plans, and book services through a single platform.

The project combines modern web development, AI-assisted planning, and streamlined booking workflows to create a luxury digital experience for brides preparing for one of the most important events of their lives.

---

## Problem Statement

Planning bridal beauty services is often fragmented and stressful.

Brides frequently face challenges such as:

* Finding trusted beauty professionals
* Comparing prices and service packages
* Managing bridal preparation timelines
* Coordinating multiple beauty services
* Tracking bookings and consultations
* Lack of centralized discovery platforms

Genzest AI addresses these challenges through a single AI-assisted platform.

---

# User Workflow

### Step 1

User visits Genzest AI.

### Step 2

User explores the platform and service offerings.

### Step 3

User opens the AI Bridal Planner.

### Step 4

User enters:

* Wedding Date
* Budget
* Skin Type
* Region

### Step 5

The platform generates recommendations and preparation guidance.

### Step 6

User browses verified artists.

### Step 7

User selects:

* Artist
* Service
* Package

### Step 8

Booking information is generated.

### Step 9

Structured WhatsApp message is prepared automatically.

### Step 10

User submits booking request.

# AI Workflow

The AI planning system follows a recommendation-based workflow.

### Input Layer

User provides:

* Wedding Timeline
* Budget Range
* Skin Profile
* Geographic Preference

### Processing Layer

The planner analyzes:

* Available preparation time
* Budget distribution
* Service priorities
* Bridal readiness milestones

### Output Layer

The system generates:

* Preparation schedule
* Budget allocation
* Recommended service categories
* Bridal action plan

This reduces planning complexity and helps users organize wedding beauty preparations efficiently.
